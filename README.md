# JMComic-Downloader

📲 **自动检查和下载禁漫天堂（JMComic）Android APK**，并将最新版发布到 GitHub Releases。

## 功能说明

- 使用 GitHub Actions 每 6 小时检测一次更新。
- 检测新版 APK 后自动下载并发布到 Release 页面。
- 使用简单，无需手动更新。
